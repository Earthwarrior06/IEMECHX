# Ultrasonic Obstacle Detection (Simulation)


HC-SR04 based obstacle alert system using Arduino.


## Features
- Detects distance using ultrasonic sensor
- LED and buzzer alert when object is closer than 20 cm
- Fully simulated using Tinkercad
